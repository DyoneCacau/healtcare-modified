### 📋 PASSO 1: EXECUTAR SQL NO SUPABASE (OBRIGATÓRIO)

**Antes de rodar o projeto**, você DEVE executar a migration SQL:

1. Acesse: https://supabase.com/dashboard
2. Vá em **SQL Editor** (menu lateral)
3. Clique em **+ New Query**
4. Abra o arquivo: `supabase/migrations/20260212_vendas_diretas.sql`
5. Copie TODO o conteúdo
6. Cole no SQL Editor
7. Clique em **RUN** ou pressione Ctrl+Enter
8. Aguarde a execução (10-30 segundos)
9. Você verá mensagens de sucesso ao final

**Sem executar este SQL, o sistema NÃO funcionará!**
